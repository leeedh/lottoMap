# 🎨 로또 명당 서비스 – UI/UX 디자인 가이드

---

## 1. 디자인 컨셉 및 원칙

### 디자인 철학

* **희망·행운·신뢰**: 사용자에게 긍정적이고 즐거운 경험 제공
* **데이터 기반 신뢰성**: 시각화와 통계를 통해 "근거 있는 명당 정보" 전달
* **간결·직관적**: 지도 중심의 단순한 인터페이스로 누구나 쉽게 탐색

### 사용자 경험 목표

* **탐색 경험**: 사용자가 전국 명당을 "지도 위에서 직접 탐험"하는 느낌
* **발견 경험**: 랭킹 기능을 통해 "재미있게 명당을 발견"
* **반응형 경험**: 모바일/데스크톱 모두 최적화된 경험

### 접근성 고려사항

* 색각 이상자를 고려한 색상 대비 (로또 빨강, 연금복권 파랑으로 명확히 구분)
* 텍스트 대비 WCAG AA 이상 확보 (배경색 대비 4.5:1 이상)
* 반응형 레이아웃: 모바일/PC 모두 사용 가능 (768px 브레이크포인트 중심)
* 아이콘 + 텍스트 병행 사용 (Lucide React 아이콘 사용)

## 2. 사용자 흐름 및 와이어프레임

### 주요 흐름

```
메인(지도 + 사이드바) → [필터/탭 전환] → 판매점 상세 정보
                      ↘ [랭킹 보기] → 판매점 상세
```

### 와이어프레임 (텍스트 표현)

**메인 페이지 (단일 페이지 애플리케이션)**

```
[데스크톱 레이아웃]
├─ 좌측 사이드바 (고정, 400px)
│  ├─ 헤더: 로고 (Trophy 아이콘) + 제목 "Lucky Map"
│  ├─ 탭: [지도 내 명당] [전체 랭킹]
│  ├─ 필터 영역
│  │  ├─ 지도 내 명당 탭: 복권 종류 필터 (로또 6/45, 연금복권)
│  │  └─ 전체 랭킹 탭: 지역 필터 (시/도, 시/군/구 드롭다운)
│  ├─ 리스트 영역
│  │  └─ 판매점 카드 (StoreCard)
│  └─ 푸터: Buy Me a Coffee 버튼
│
└─ 우측 지도 영역 (flex-1)
   └─ Kakao Map + 커스텀 마커

[모바일 레이아웃]
├─ 상단 오버레이
│  ├─ 로고 카드 + 커피 후원 버튼
│  └─ 플로팅 필터 칩 (로또 6/45, 연금복권)
│
├─ 지도 영역 (전체 화면)
│  └─ Kakao Map + 커스텀 마커
│
└─ 하단 플로팅 버튼
   └─ "명당 리스트 보기" 버튼 (사이드바 드로어 열기)

[모바일 사이드바 드로어]
└─ 전체 화면 오버레이
   ├─ 헤더: 닫기 버튼 + 로고 + 제목
   ├─ 탭: [지도 내 명당] [전체 랭킹]
   ├─ 필터 영역
   ├─ 리스트 영역
   └─ 푸터: Buy Me a Coffee 버튼
```

**판매점 상세 정보 (인포윈도우 및 카드 확장)**

```
[인포윈도우 (지도 마커 클릭)]
├─ 판매점명 (18px Bold)
├─ 주소 (12px)
├─ 당첨 통계 (로또 1등, 로또 2등, 연금복권)
└─ 최근 당첨 이력 (회차, 등수, 방법, 날짜)

[카드 확장 (사이드바 카드 클릭)]
├─ 헤더: 판매점명 + 카테고리 뱃지 + 확장 아이콘
├─ 주소 (MapPin 아이콘)
├─ 당첨 통계 그리드 (로또 1등, 로또 2등, 연금복권)
└─ 최근 당첨 인증 (확장 시 표시)
   └─ 회차별 이력 리스트
```

## 3. UI 컴포넌트 가이드 (현재 구현)

### 버튼

* **필터 버튼 (필터 칩)**
  * 모서리: rounded-full
  * 활성화: 배경색 (로또: `bg-lotto`, 연금: `bg-pension`), 흰색 텍스트, ring-2
  * 비활성화: 흰색 배경, 회색 텍스트, border
  * 크기: px-4 py-2, text-xs font-bold

* **하단 플로팅 버튼 (모바일)**
  * 배경: bg-gray-900, text-white
  * 모서리: rounded-full
  * 그림자: shadow-xl
  * 크기: pl-5 pr-6 py-3.5

* **Buy Me a Coffee 버튼**
  * 배경: #FFDD00 (노란색)
  * 모서리: rounded-xl
  * 텍스트: font-bold

### 필터 칩 (모바일)

* **활성화 상태**
  * 로또: bg-lotto (red-500 #ef4444), text-white, border-red-600, ring-2 ring-red-100
  * 연금: bg-pension (blue-500 #3b82f6), text-white, border-blue-600, ring-2 ring-blue-100
* **비활성화 상태**
  * bg-white, text-gray-500, border-gray-200

### 지도 마커 (커스텀 마커)

* **로또 6/45 마커**
  * 배경색: #ef4444 (red-500)
  * 테두리: 4px solid #fecaca (red-200)
  * 아이콘: 복권 아이콘 (SVG, 흰색)
  * 크기: 32px (기본), 40px (3회 이상 당첨)
  * 배지: 3회 이상 당첨 시 검은색 배지에 횟수 표시

* **연금복권 마커**
  * 배경색: #3b82f6 (blue-500)
  * 테두리: 4px solid #bfdbfe (blue-200)
  * 아이콘: 별 아이콘 (SVG, 흰색)
  * 크기: 32px (기본), 40px (3회 이상 당첨)
  * 배지: 3회 이상 당첨 시 검은색 배지에 횟수 표시

* **선택된 마커**
  * 지도에서 해당 위치로 이동 (panTo)
  * 인포윈도우 표시

### 카드 (판매점 카드 - StoreCard)

* **기본 스타일**
  * 배경: bg-white
  * 모서리: rounded-xl (12px)
  * 그림자: shadow-sm, hover:shadow-md
  * 테두리: border-l-4 (로또: border-l-lotto, 연금: border-l-pension)

* **선택 상태**
  * ring-2 ring-gray-900
  * bg-gray-50
  * transform scale-[1.01]

* **구성 요소**
  * 헤더: 판매점명 (text-lg font-bold) + 주소 (text-xs) + 카테고리 뱃지 + 확장 아이콘
  * 당첨 통계 그리드 (grid-cols-3):
    * 로또 1등: bg-red-50, border-red-100
    * 로또 2등: bg-orange-50, border-orange-100
    * 연금복권: bg-blue-50, border-blue-100
  * 확장 영역: 최근 당첨 이력 (회차, 등수, 방법, 날짜)

### 사이드바 (Sidebar)

* **데스크톱**
  * 위치: fixed, left-0, width: 400px
  * 배경: bg-white
  * 그림자: shadow-2xl

* **모바일**
  * 드로어 형태: 전체 화면 오버레이
  * z-index: 1002
  * transform: translate-x-0 (열림), -translate-x-full (닫힘)

* **탭**
  * border-b-2 (활성: border-gray-900, 비활성: border-transparent)
  * 텍스트: text-sm font-bold

* **필터 영역**
  * 배경: bg-gray-50
  * border-b: border-gray-100

### 인포윈도우 (지도 마커 클릭 시)

* **스타일**
  * min-width: 200px
  * padding: 12px
  * font-family: system-ui, -apple-system, sans-serif

* **구성 요소**
  * 판매점명 (18px Bold)
  * 주소 (12px, gray-600)
  * 당첨 통계 (flex, bg-gray-50, rounded-lg)
  * 최근 당첨 이력 (리스트, max-height: 120px, overflow-y: auto)

## 4. 인터랙션 및 애니메이션 지침

* **마커 클릭**
  * 인포윈도우 표시
  * 지도에서 해당 위치로 이동 (panTo)
  * 카드 클릭 시 지도 이동 및 선택 표시

* **카드 확장/축소**
  * grid-rows-[1fr] (확장) / grid-rows-[0fr] (축소)
  * transition-all duration-300 ease-in-out
  * 확장 아이콘 회전 (ChevronDown, rotate-180)

* **탭 전환**
  * border-b-2 색상 변경 (transition-colors)
  * 필터 영역 내용 변경

* **사이드바 드로어 (모바일)**
  * transform transition-transform duration-300 ease-in-out
  * translate-x-0 (열림) / -translate-x-full (닫힘)

* **필터 버튼**
  * active:scale-95 (클릭 시)
  * transition-all

## 5. 비주얼 스타일 가이드 (현재 구현)

### 색상 팔레트

* **Primary (브랜드)**
  * 로또: `lotto` (#ef4444, red-500)
  * 연금복권: `pension` (#3b82f6, blue-500)

* **Secondary**
  * 노랑: #FFDD00 (Buy Me a Coffee 버튼)

* **Background**
  * 흰색: #FFFFFF (카드, 사이드바)
  * 연회색: #F9F9F9 (bg-gray-50, 필터 영역)
  * 회색: #F1F1F1 (스크롤바 트랙)

* **Text**
  * 진회색: #111827 (gray-900, 제목)
  * 회색: #6B7280 (gray-500, 보조 텍스트)
  * 연회색: #9CA3AF (gray-400, 캡션)

* **State colors**
  * 성공: #4CAF50 (green-500)
  * 오류: #F44336 (red-600)
  * 경고: #FF9800 (orange-500)

### 타이포그래피

* **폰트**: 시스템 폰트 (system-ui, -apple-system, sans-serif)
* **H1**: text-2xl font-black (24px, 사이드바 제목)
* **H2**: text-lg font-bold (18px, 판매점명)
* **H3**: text-sm font-bold (14px, 섹션 제목)
* **본문**: text-base (16px), text-sm (14px)
* **캡션/보조**: text-xs (12px), text-[10px] (10px, 뱃지)

### 아이콘 스타일

* **라이브러리**: Lucide React
* **스타일**: 선형 아이콘 (Line style, stroke)
* **크기**: 
  * 기본: 16px, 20px, 24px
  * 작은 아이콘: 10px, 12px, 14px

### 이미지 사용

* 판매점 실제 사진은 제공하지 않음 (저작권 문제)
* 지도 + 주소 기반 시각화
* 아이콘: Lucide React SVG 아이콘
